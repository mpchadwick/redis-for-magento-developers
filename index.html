The basics

What is Redis?

Redis is an open source (BSD licensed), in-memory data structure store, used as a database, cache and message broker

https://redis.io/

https://github.com/antirez/redis

It's on GitHub!

http://antirez.com/

https://twitter.com/antirez

It's was created by Salvatore Sanfilippo (a.k.k antirez). Check out his website and follow him on Twitter.

Why Redis?

  • In memory == FAST
  • Many advantages over memcached
  • Most popular cache backend for Magento
    • ## Basic Usage ```bash # Connect $ redis-cli -h HOST -p PORT # Human readable information and statistics 127.0.0.1:6379> INFO OK # Get a config value 127.0.0.1:6379> CONFIG GET maxmemory OK # Flushing a database 127.0.0.1:6379> SELECT 0 OK 127.0.0.1:6379> FLUSHDB OK ```
    ## Be CAREFUL! ```bash # Show all keys - May cause service interruption 127.0.0.1:6379> KEYS * OK # Whoops - Cleared all the sessions 127.0.0.1:6379> FLUSHALL OK ```

    How Magento Uses Redis

  • "Object" cache
  • Session storage
  • Page cache (only recommended for Magento 1)
